# SkillForge - Android  Assignment

## AI-Assisted Development

I used AI as a development assistant throughout this assignment to improve productivity, validate implementation ideas, and speed up debugging. Every AI-generated suggestion was reviewed, tested, and modified before being integrated into the project.

### AI Tools Used

- ChatGPT
- Gemini

### Example Prompts

**Prompt 1**
> Build a Jetpack Compose UI that closely matches the provided SkillForge design while keeping the code clean and reusable.

**Prompt 2**
> Suggest improvements for MVVM architecture, state management, API integration, and Compose UI organization for this Android application.

### What AI Got Right

AI significantly accelerated development by helping with project structure, Retrofit setup, Compose layouts, API integration, debugging, and code organization. It was especially useful for generating repetitive boilerplate code and suggesting implementation approaches.

### What AI Got Wrong

Some generated UI layouts and navigation suggestions did not perfectly match the assignment requirements or Compose behavior. Instead of using them directly, I manually reviewed the 
code, adjusted the implementation, fixed the issues, and verified the final behavior against the provided design and API.

### My Development Workflow

- Analyzed the assignment requirements.
- Created the project structure and data models.
- Integrated the provided REST API using Retrofit.
- Built the UI with Jetpack Compose.
- Used AI to validate implementation ideas and troubleshoot issues.
- Reviewed, tested, and refined all generated code before finalizing the application.
